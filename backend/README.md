# Проект Mesto / бэкенд


## Описание

Сервер на Express.js c базой данных Mongo DB. Бэкенд для проекта Mesto — упрощенного аналога Instagram. 

## Запуск проекта

`npm run start` — запускает сервер   
`npm run dev` — запускает сервер с hot-reload
